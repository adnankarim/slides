** About GNU Emacs

   - Emacs has been around since 1970's
   - Originally written by [[http://www.stallman.org/][Richard Stallman]] & [[http://en.wikipedia.org/wiki/Guy_L._Steele,_Jr.][Guy L. Steele, Jr.]]
   - It's core is written in *C*
