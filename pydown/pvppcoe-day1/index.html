RedHat

* Recommended for servers

* Highly stable

* Linux Admins/Advance users

* Enterprise use :(

CentOS

* Based on RedHat's source code

* Community based

* Best alternative for RedHat

Fedora

* Based on RedHat's source code

* Community based

* Alternative for RedHat
